  • SI57X_SCL [read/write]: SI57X_SCL
    GPIO controlling the state of the SCL line of the Silabs Si57p oscillator on the ADC mezzanine.
  • SI57X_SDA [read/write]: SI57X_SDA
    GPIO controlling the state of the SDA line of the Silabs Si57p oscillator on the ADC mezzanine.
  • SPI_CS_ADC [read/write]: SPI_CS_ADC
    GPIO controlling the state of the Chip Select line of the LTC2175 ADC on the ADC mezzanine
  • SPI_SCK [read/write]: SPI_SCK
    GPIO controlling the state of the SCK line of the LTC2175 ADC on the ADC mezzanine
  • SPI_MOSI [read/write]: SPI_MOSI
    GPIO controlling the state of the MOSI line of the LTC2175 ADC on the ADC mezzanine
  • SPI_MISO [read-only]: SPI_MISO
    GPIO with the state (input) of the MISO line of the LTC2175 ADC on the ADC mezzanine
  • TM_LOCK_EN [read/write]: TM_LOCK_EN
    write 1: enable locking of the ADC clock to WR. Enable only after Si57x has been initialized.
  • TM_LOCKED [read-only]: TM_LOCKED
    When 1, the ADC clock is locked to WR.
  • TM_TIME_VALID [read-only]: TM_TIME_VALID
    when 1, the WR Core is synchronized with the WR network

  • FULL [read-only]: FIFO full flag
    1: FIFO 'ADC Data FIFO' is full
    0: FIFO is not full
  • EMPTY [read-only]: FIFO empty flag
    1: FIFO 'ADC Data FIFO' is empty
    0: FIFO is not empty
  • CLEAR_BUS [write-only]: FIFO clear
    write 1: clears FIFO 'ADC Data FIFO
    write 0: no effect
  • USEDW [read-only]: FIFO counter
    Number of data records currently being stored in FIFO 'ADC Data FIFO'
